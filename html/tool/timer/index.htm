很高兴你能来到这里，这说明你有一颗善于探索的心，总希望了解事物的本质、原理。

Aero Timer Web 是一个完完全全的 JavaScript + HTML 应用，因此所有的代码都可以通过 “鼠标右键”->"查看网页源代码" 得到。

定时器核心代码在 core.js 中，style.css 定义了页面样式，代码中 jQuery 缩写为 $j ，剩下的就该你自己去探索了。
